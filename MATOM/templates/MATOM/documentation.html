Documentation: MARS tutorial

This documentation provides a guide of how to optimally utilize the tools provided in this web server. MARS is set-up specifically for the users interested in obtaining rank information the motifs available in  the database, as well as to rank motif determine the performance of their own motif.

The minimum required information is a transcription factor name to retrieve ranked motifs. For each tool, the default parameters should provide the best results. Use the decision diagram below to choose the appropriate tool for your task.


This is a data independent  approach, which uses the principle of consistency to determine the quality and rank the available motifs. We employ two motif assessment techniques to achieve this.

  • Tomtom: This is the default and has been shown to provide the best results.
  • FISim: An alternative approach implemented

Motifs are used to score positive and negative sequences and the quality and ranks of the motifs determined by AUC, MNCP and correlation statistics. This approach implements a number of scoring functions. Follow link for more details on these techniques. In the approach the user has to make the following choices

  1. Provide a TF name (Required). This can be the only details needed for a complete analysis, if the corresponding data are available (1)
  2. Do you want to rank your own motif against the available motifs? 
    • Then paste or upload motif in meme format (1)
  3. Choose the experimental evidence to use, each allows one to upload own data:
    1. ChIP-seq data
    2. PBM data
  4. Select the scoring function to use. 
  5. Run the analysis and wait for the results

  1. Choose enrichment technique to use
  2. Enter TF name
  3. Upload or paste motifs if need be
  4. Uploaded own benchmark data
  5. Run analysis and wait for results

The process of motif assessment can be summarized as (1) User uploads a motif in MEME format and enters its TF name. (2) Motifs and test sequences linked to the TF are extracted from the database. (3) Motifs can be ranked, by comparison, used to score test sequences (rank by score) or its enrichment determined using CentriMo (rank by enrichment). (4) The results are visualized interactively, (5) with additional information like motif length, Information content and logo. The cluster-gram offers additional information on the motif or test data clustering. In the end, the user can download ranked motifs in MEME format, as well as raw data for further analysis.

We use as input meme formatted motifs . However, we do provide scripts that can convert motif available in other formats to meme.

We accept sequences in bed and narrow peak formats. The negative sequences are obtained from hg19 human genome.

We accept an upload of protein binding micro-array intensity data.

We plan to include more benchmark data. Please let us know if there are data you require, and we'll do our best to include them.
